## 🎯 Deployment Instructions (Simple 4 Steps)

### Step 1: Install Vercel CLI
```bash
npm install -g vercel
```

### Step 2: Deploy to Vercel (FREE)
```batch
cd vercel-sidecar
.\deploy.bat
```
*(Follow the prompts - creates account if needed)*

### Step 3: Configure Environment
- **Vercel Dashboard**: Add `PLANNER_SIDECAR_TOKEN` environment variable
- **InfinityFree .htaccess**: Add these lines:
  ```apache
  SetEnv PLANNER_SIDECAR_URL https://your-project.vercel.app/api/planner
  SetEnv PLANNER_SIDECAR_TOKEN your-secure-token
  ```

### Step 4: Test Everything
```batch
.\test.ps1
```
